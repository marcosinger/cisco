# Cisco

A golang project to return the status of some services.

## Installation

```go get -v github.com/marcosinger/cisco```

Once the get completes, you should find your new cisco executable sitting inside $GOPATH/bin/.

## Usage

```bash
  cisco

  [Heroku Status]
  Production: [ok]
  Development: [ok]
```
